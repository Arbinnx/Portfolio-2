About me

The developer
behind the work

I started learning web development because making things you can actually see and use in a browser felt real in a way nothing else did. HTML, CSS, JavaScript — I built small things, then bigger ones.

Today I focus on the frontend. I write clean HTML, style things properly with CSS and Bootstrap, and make it interactive with JavaScript. I use Git for everything I build. I'm also working through PHP and Laravel to start offering backend features too.

I'm still learning — and I know that. But I take the work seriously, ask questions when I'm stuck, and I don't pretend to know more than I do. That honesty saves everyone time.
